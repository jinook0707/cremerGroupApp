1. SHORT DESCRIPTION

- 'cremerGroupApp' is an assortment of programs and scripts to facilitate 
experiments on ant species and data analysis, conducted in cremer-group 
in Institute of Science and Technology Austria. 


2. Dependencies

- Python 3.x

- Python package dependencies 
    - openCV
    - wxPython
    - sciPy
    - MatPlotLib
    - decord
    - tsmoothie
    - scikit-video
    - pyueye (optional)
    - pyaudio (optional)

* All programs were tested in Ubuntu Linux 18.04 and Windows 10.
